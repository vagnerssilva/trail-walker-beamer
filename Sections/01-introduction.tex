\begin{frame}[t]{Introdução} 
    O projeto consiste em desenvolver um robô de pequeno porte que se desloca sobre dois pés. O robô deve ser capaz de se locomover e desviar de obstáculos em um ambiente.
   
        \begin{columns}[c]
            \column{.6\textwidth}
                \begin{itemize}
                    \item Desenvolver algoritmos utilizando ROS2
                    \item Implementar visão computacional
                    \item Simular o sistema robótico
                    \item Realizar demonstração do sistema
                    \item Desenvolvimento de artigos científicos
                \end{itemize}
            \column{.25\textwidth}
                \includegraphics[width=1.5\textwidth]{walker}
        \end{columns}

\end{frame}
